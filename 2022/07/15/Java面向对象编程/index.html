基础内容

基本数据类型

byte：

  • 最大值：127（2^7-1）

  • 最小值：128（-2^7）

  • 默认值：0

  • 例子：byte a = 100;

short：

  • 最大值：32767（2^15 - 1）
  • 最小值：- 32768（-2^15）
  • 默认值：0
  • 例子：short r = 2000

int：

  • 最大值是 2,147,483,647（2^31 - 1）
  • 最小值是 -2,147,483,648（-2^31）
  • 默认值是 0 ；
  • 例子：int a = 100000

long：

  • 最大值：9,223,372,036,854,775,807（2^63 -1）
  • 最小值：-9,223,372,036,854,775,808（-2^63）
  • 默认值：0L（L是不分大小写的，但是写成大写不容易看错）
  • 例子：long a = 10L

float：

  • 单精度32位浮点数
  • 例子：float num = 233.45f

double：

  • 双精度64位浮点数
  • 例子：double num = 7

boolean：

  • 布尔类型，用来表示真假
  • 取值：true和false
  • 例子：boolean flag = true

char：

  • 字符类型
  • char a = ‘A’
  • java中字符串不是字符数组，它是引用类型
